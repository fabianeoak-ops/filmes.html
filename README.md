Galeria de Filmes Favoritos

Descrição

Projeto desenvolvido como atividade prática da disciplina de Desenvolvimento WEB, utilizando HTML para criar uma galeria de filmes favoritos. 
As imagens são armazenadas localmente e cada filme possui um link para sua página correspondente na Wikipédia.

Objetivos
Trabalhar com elementos multimídia em páginas web.
Inserir imagens locais em HTML.
Utilizar hyperlinks externos.
Organizar conteúdo em uma página web simples.

Jurassic Park

Como Executar
Faça o download ou clone o repositório.
Certifique-se de que todos os arquivos estejam na mesma pasta.
Abra o arquivo filmes.html em qualquer navegador.

Funcionalidades
Exibição de imagens dos filmes favoritos.
Acesso às páginas da Wikipédia por meio de links.
Link para retorno ao topo da página.
